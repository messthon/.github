## Hi there 👋

[EN]

Messthon is a private messenger written in Python by developer <a href="https://github.com/Kolyadual/">Kolyadual</a>. The messenger combines Telegram, Discord, 
Matrix, WhatsApp, Tox, and other popular communication tools into a single app, where everything works without restrictions or slowdowns or VPN. This is especially true in Russia.

[RU]

Messthon — это приватный мессенджер, написанный на Python разработчиком <a href="https://github.com/Kolyadual/">Kolyadual</a>. Мессенджер объединяет Telegram, Discord,
Matrix, WhatsApp, Tox и другие популярные средства связи в одном приложении, где всё работает без ограничений, замедлений или VPN. Это особенно актуально в России.
